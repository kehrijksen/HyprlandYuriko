# Hyprland Yuriko
My Hyprland configuration featuring "Yuriko, the Tiger's Shadow"

![preview](preview.png)

## Installation
```
sudo pacman -S hyprland hyprpaper hyprlock wofi waybar swaync hyprshot satty alacritty fish fastfetch ttf-nerd-fonts-symbols otf-font-awesome
```

```
git clone git@github.com:kehrijksen/HyprlandYuriko.git
```

```
cp -R HyprlandYuriko/ ~/
```
