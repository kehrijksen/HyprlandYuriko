# Hyprland Yuriko
My Hyprland configuration featuring "Yuriko, the Tiger's Shadow"

## Installation
`pacman -S hyprland hyprpaper hyprlock wofi waybar swaync alacritty fish fastfetch`

`git clone git@github.com:kehrijksen/HyprlandYuriko.git`

`cp -R HyprlandYuriko/ ~/`

## Programs
1. hyprland
1. hyprpaper
1. hyprlock
1. wofi
1. waybar
1. swaync
1. alacritty
1. fish
1. fastfetch
